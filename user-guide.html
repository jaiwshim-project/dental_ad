메뉴 구조 이해하기

상단 네비게이션 메뉴

모든 페이지 상단에서 다음 메뉴를 통해 빠르게 이동할 수 있습니다:

  • 홈: 메인 페이지로 이동
  • 생성시작: 새로운 광고 콘텐츠 생성
  • 샘플보기: 저장된 샘플 케이스 확인
  • 내 프로필: 저장한 케이스 목록 관리

홈페이지 메뉴 카드

  • 🚀 새로 시작하기: 치과 정보를 입력하여 맞춤 콘텐츠 생성
  • 📂 저장된 케이스: 이전에 저장한 케이스 목록
  • 🔧 데이터 초기화: 저장된 데이터 확인 및 삭제
🚀

시작하기

1

샘플 케이스 체험하기

처음 사용하시는 경우, 두 가지 방법으로 샘플을 확인할 수 있습니다:

  • 홈페이지 중앙 배너: "🎯 샘플 케이스 보기" 버튼 클릭
  • 상단 메뉴: "샘플보기" 메뉴를 통해 여러 케이스 탐색
💡 TIP: 화이트서울치과 임영일 원장의 실제 진단 결과를 기반으로 한 샘플로, 어떤 정보가 필요하고 어떤 결과가 나오는지 미리 파악할 수 있습니다.
2

새 케이스 생성하기

홈페이지 또는 상단 메뉴에서 생성을 시작할 수 있습니다:

  • 홈페이지 메뉴 카드: "🚀 새로 시작하기" 클릭
  • 상단 메뉴: "생성시작" 클릭
✍️

정보 입력하기

1

기본 정보 입력

  • 치과 이름: 귀원의 정식 명칭을 입력합니다
  • 원장 이름: 대표 원장님의 성함을 입력합니다
  • 위치: 치과의 지역 정보를 입력합니다 (예: 서울 강남구)
  • 전문 분야: 주력 진료 분야를 입력합니다 (예: 임플란트, 교정)
2

매출 단계 선택

현재 치과의 매출 상황을 가장 잘 나타내는 단계를 1순위와 2순위로 선택합니다.

  • 1형: 수익 정체·감소
  • 2형: 수익 유지·불안정
  • 3형: 원장 과부하
  • 4형: 수익 안정·확장 준비
  • 5형: 고수익·브랜드형
3

원장 스타일 선택

원장님의 진료 스타일을 가장 잘 나타내는 유형을 1순위와 2순위로 선택합니다.

  • A형: 진료·기술형 - 치료 자체에 집중
  • B형: 학술·연구형 - 최신 기술과 연구에 관심
  • C형: 사람·감성형 - 환자와의 교감 중시
  • D형: 수익·경영형 - 효율적 운영 중시
  • E형: 환자·신뢰형 - 장기적 관계 구축
4

DF-AI 신뢰 점수 입력

0~100점 범위에서 각 항목의 점수를 입력합니다. 점수가 높을수록 해당 영역이 강합니다.

  • 공감: 환자의 감정과 상황을 이해하는 능력
  • 이해: 치료 내용을 명확하게 설명하는 능력
  • 결정 자율성: 환자에게 선택권을 부여하는 정도
  • 가치 전달: 치료의 가치를 효과적으로 전달하는 능력
  • 신뢰 관계: 장기적 신뢰 관계를 구축하는 능력
  • 공간/감성: 치과 환경과 분위기
💡 TIP: 점수는 솔직하게 평가하세요. 낮은 점수도 그에 맞는 최적의 광고 전략을 생성합니다.
5

추가 정보 입력 (선택)

  • 치과 철학: 귀원만의 진료 철학이나 가치관
  • 차별화 포인트: 다른 치과와 구별되는 특별한 점
  • 선호 채널: 주로 활용하고 싶은 마케팅 채널
6

비밀번호 설정

케이스를 안전하게 보호하기 위한 비밀번호를 설정합니다.

  • 4자리 이상의 비밀번호를 입력하세요
  • 저장된 케이스를 다시 볼 때 이 비밀번호가 필요합니다
  • 비밀번호를 잊어버리면 해당 케이스에 접근할 수 없으니 주의하세요
💡 TIP: 비밀번호는 안전하게 기억해두세요. 분실 시 케이스에 접근할 수 없습니다.
✨

결과 확인 및 활용하기

1

5가지 콘텐츠 유형 탐색

결과 페이지에서 5개의 탭을 통해 다양한 유형의 광고 콘텐츠를 확인할 수 있습니다.

  • 공감형: 환자의 감정과 공감하는 콘텐츠
  • 이해형: 치료 정보를 명확하게 설명하는 콘텐츠
  • 결정형: 환자의 선택권을 강조하는 콘텐츠
  • 가치전달형: 치료의 가치를 강조하는 콘텐츠
  • 신뢰형성용: 장기적 신뢰를 구축하는 콘텐츠
2

콘텐츠 분량 조절

상단의 버튼을 통해 콘텐츠 분량을 조절할 수 있습니다.

  • 기본형: 짧고 간결한 메시지 (SNS에 적합)
  • 중간형: 적당한 분량 (블로그 포스팅에 적합)
  • 장문형: 상세한 내용 (홈페이지, 브랜드 스토리에 적합)
3

광고 허용 레벨 조절

광고의 공격성 정도를 L0~L5까지 조절할 수 있습니다.

  • L0~L1: 매우 부드러운 톤 (신뢰 구축 단계)
  • L2~L3: 중간 톤 (일반적인 광고)
  • L4~L5: 적극적인 톤 (캠페인, 프로모션)
✏️

콘텐츠 수정하기

1

수정 버튼 클릭

각 광고 콘텐츠 카드의 "✏️ 수정" 버튼을 클릭합니다.

2

수정 요청사항 입력

팝업창에서 수정하고 싶은 내용을 자유롭게 입력합니다.

  • "더 친근한 톤으로 바꿔주세요"
  • "50대 여성 환자를 타겟으로 수정"
  • "할인 이벤트 정보를 추가해주세요"
  • "임플란트에 대한 설명 강화"
3

수정 적용

"수정 적용" 버튼을 클릭하면 요청사항이 콘텐츠에 표시됩니다.

💡 TIP: 요청사항은 핑크색 박스로 표시되어 실제 사용 시 참고할 수 있습니다.

저장 및 관리하기

📋

복사하기

콘텐츠를 클립보드에 복사하여 바로 사용할 수 있습니다

💾

파일로 저장

텍스트 파일로 다운로드하여 보관할 수 있습니다

전체 결과를 케이스로 저장하여 나중에 다시 확인할 수 있습니다

📚

저장된 케이스 관리

홈페이지 메뉴 또는 상단의 "내 프로필" 메뉴에서 저장한 케이스들을 확인할 수 있습니다

🔄

새로운 광고 콘텐츠 생성하기

이미 저장된 프로필이 있다면, 처음부터 다시 입력할 필요 없이 기존 정보를 활용해서 새로운 광고 콘텐츠를 빠르게 생성할 수 있습니다.

1

내 프로필로 이동

상단 메뉴에서 "내 프로필"을 클릭합니다.

2

프로필 카드 찾기

저장된 프로필 목록에서 원하는 치과의 프로필 카드를 찾습니다. 각 카드에는 고유 넘버와 치과 이름이 표시되어 있습니다.

  • 고유 넘버: 각 프로필마다 자동으로 부여된 식별 번호
  • 치과 정보: 치과명, 원장명, 위치 등 기본 정보 표시
3

생성하기 버튼 클릭

원하는 프로필 카드에서 "생성하기" 버튼을 클릭합니다.

이 경우, 처음 생성할 때 설정했던 4자리 비밀번호 숫자를 입력하는 창이 나타납니다. 비밀번호가 일치하는 경우에만 기본 정보 입력란으로 이동합니다.

⚠️ 주의: 비밀번호를 잊어버린 경우 해당 프로필에 접근할 수 없습니다. 비밀번호를 정확하게 입력해주세요.
4

자동 입력된 정보 확인

입력 페이지로 이동하면 저장된 프로필의 모든 기본 정보가 자동으로 입력된 상태로 나타납니다.

  • 치과 이름, 원장 이름, 위치, 전문 분야
  • 매출 단계 선택 (1순위, 2순위)
  • 원장 스타일 선택 (1순위, 2순위)
  • DF-AI 신뢰 점수 (6개 항목)
  • 추가 정보 (치과 철학, 차별화 포인트, 선호 채널)
💡 TIP: 자동 입력된 정보를 그대로 사용하거나, 필요한 부분만 수정한 후 새로운 광고 콘텐츠를 생성할 수 있습니다. 이렇게 하면 매번 처음부터 입력하지 않아도 됩니다!
5

콘텐츠 생성

정보를 확인하거나 수정한 후 "광고 콘텐츠 생성하기" 버튼을 클릭하면 새로운 광고 콘텐츠가 생성됩니다.

자주 묻는 질문 (FAQ)

Q. 생성된 콘텐츠를 상업적으로 사용해도 되나요?
A. 네, 생성된 모든 콘텐츠는 귀원의 마케팅 목적으로 자유롭게 사용하실 수 있습니다.
Q. 데이터는 어디에 저장되나요?
A. 모든 데이터는 브라우저의 localStorage에 저장되어 외부로 전송되지 않습니다. 개인정보 보호를 위해 브라우저를 닫아도 데이터가 유지됩니다.
Q. 여러 치과의 케이스를 관리할 수 있나요?
A. 네, 각 케이스를 저장하면 홈페이지의 "📂 저장된 케이스" 메뉴 또는 상단의 "내 프로필"에서 여러 치과의 케이스를 관리할 수 있습니다.
Q. 생성된 콘텐츠가 마음에 들지 않으면 어떻게 하나요?
A. 콘텐츠 분량과 광고 허용 레벨을 조절하거나, "✏️ 수정" 버튼을 통해 원하는 방향으로 수정 요청을 할 수 있습니다. 또한 다른 콘텐츠 유형 탭을 확인해보세요.
Q. 샘플 케이스는 실제 치과인가요?
A. 네, 화이트서울치과 임영일 원장의 실제 진단 결과를 기반으로 한 샘플입니다.
Q. 이전에 저장한 케이스를 어떻게 찾나요?
A. 상단 메뉴의 "내 프로필" 또는 홈페이지의 "📂 저장된 케이스" 카드를 클릭하면 저장된 모든 케이스 목록을 확인할 수 있습니다.
Q. 상단 네비게이션 메뉴가 안 보여요.
A. 모든 페이지 최상단에 고정된 네비게이션 바가 있습니다. 페이지를 스크롤해도 항상 상단에 표시됩니다. 모바일에서는 메뉴가 축소되어 표시될 수 있습니다.
